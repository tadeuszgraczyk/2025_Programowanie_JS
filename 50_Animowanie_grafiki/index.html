Animowanie grafiki na stronach internetowych to jedna z najbardziej kreatywnych i wizualnie satysfakcjonujących umiejętności, które można osiągnąć za pomocą JavaScriptu. Dzięki animacjom strony stają się bardziej interaktywne, atrakcyjne i angażujące dla użytkowników. JavaScript oferuje wiele sposobów na tworzenie animacji — od prostych zmian stylów CSS po zaawansowane animacje kanwy (Canvas) czy biblioteki takie jak GSAP czy Three.js.

Podstawowe techniki animacji w JavaScript

1. Animacje CSS + JavaScript

Najprostszy sposób na animowanie elementów to połączenie CSS z JavaScriptem. Można używać właściwości takich jak transition lub animation w CSS, a JavaScriptem sterować ich uruchamianiem.

Przykład: Płynna zmiana koloru tła

<div id="box" style="width: 100px; height: 100px; background-color: red; transition: background-color 1s;">
</div>
<button onclick="animateBox()">Zmień kolor</button>
<script>
function animateBox() {
const box = document.getElementById("box");
box.style.backgroundColor = "blue";
}
</script>

2. Animacje z użyciem requestAnimationFrame

Funkcja requestAnimationFrame pozwala na tworzenie płynnych animacji, synchronizując je z odświeżaniem ekranu. Jest to bardziej wydajne rozwiązanie niż używanie setInterval lub setTimeout.

Przykład: Animacja przesuwania elementu

<div id="movingBox" style="width: 50px; height: 50px; background-color: green; position: absolute; left: 0;">
</div>
<script>
const box = document.getElementById("movingBox");
let position = 0;
function animate() {
position += 1;
box.style.left = position + "px";
if (position < 300) {
requestAnimationFrame(animate);
}
}
animate();
</script>

3. Animacje na kanwie (Canvas)

Element <canvas> pozwala na rysowanie i animowanie grafiki 2D i 3D za pomocą JavaScriptu. Jest to idealne rozwiązanie do tworzenia gier, wizualizacji danych czy zaawansowanych efektów wizualnych.

Przykład: Animacja poruszającego się koła

<canvas id="myCanvas" width="400" height="200" style="border:1px solid #000;">
</canvas>
<script>
const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");
let x = 0;
function drawCircle() {
ctx.clearRect(0, 0, canvas.width, canvas.height); // Wyczyść kanwę
ctx.beginPath();
ctx.arc(x, 100, 20, 0, Math.PI * 2); // Narysuj koło
ctx.fillStyle = "blue";
ctx.fill(); x += 1;
if (x < canvas.width + 20) {
requestAnimationFrame(drawCircle);
}
}
drawCircle();
</script>

Biblioteki do animacji

Jeśli chcesz tworzyć bardziej zaawansowane animacje, warto skorzystać z gotowych bibliotek:

  • GSAP (GreenSock Animation Platform) – potężne narzędzie do tworzenia zaawansowanych animacji.
  • Three.js – biblioteka do tworzenia grafiki 3D w przeglądarce.
  • Anime.js – lekka biblioteka do tworzenia animacji CSS, SVG i DOM.

Dobre praktyki

  • Optymalizuj wydajność – unikaj zbyt skomplikowanych animacji, które mogą spowolnić stronę.
  • Używaj requestAnimationFrame – zamiast setInterval lub setTimeout, aby zapewnić płynność animacji.
  • Testuj na różnych urządzeniach – upewnij się, że animacje działają płynnie zarówno na komputerach, jak i na urządzeniach mobilnych.
